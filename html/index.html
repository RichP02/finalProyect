API y Visualización de Datos

En este proyecto, asumiremos el papel de un desarrollador apasionado por los datos que está creando una API que se ejecutará localmente. Además, se creará un archivo HTML que mostrará gráficos basados en la opción seleccionada por el usuario.

Este proyecto es una excelente oportunidad para demostrar habilidades tanto en el desarrollo de APIs como en la visualización de datos.

Objetivo

El objetivo es crear una API local y un archivo HTML que pueda representar de manera efectiva los datos a través de gráficos, dependiendo de la opción seleccionada por el usuario.

¿Cómo lo haremos?

Desarrollaremos una API que se ejecutará localmente y proporcionará los datos necesarios para nuestras visualizaciones. Luego, crearemos un archivo HTML que muestre gráficos basados en la opción seleccionada por el usuario. Los gráficos estarán diseñados para proporcionar una comprensión clara y efectiva de los datos.
