API y Visualización de Datos

En este proyecto, asumiremos el papel de un desarrollador apasionado por los datos que creará un archivo HTML que mostrará distintos gráficos.

El objetivo es crear un archivo HTML que pueda representar de manera efectiva los datos a través de gráficos.

¿Cómo lo haremos?

Vamos a crear diversas páginas donde se exhibirán diversas gráficas generadas en Python, utilizando las bibliotecas pandas, numpy y matplotlib. El propósito es presentar una variedad de gráficos mientras se recopila información crucial para la toma de decisiones e análisis de datos.

Problemática

Para lograr la visualización de las gráficas en HTML, fue necesario recurrir a la implementación de un diccionario en Python. Esta decisión surgió debido a desafíos encontrados al intentar que la herramienta de Pyscript leyera correctamente un archivo CSV. El diccionario en Python se convirtió en una solución efectiva para superar estos obstáculos y permitir la integración exitosa de las gráficas en la interfaz web. Este enfoque demostró ser crucial para garantizar la correcta interpretación y presentación de los datos en el formato deseado.
